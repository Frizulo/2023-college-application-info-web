# 2023 College Application Info Web

本專案為大學一年級「互動式網頁技術」課程之期末專題，使用 Django 框架開發。

## 專案簡介
本專題以「大學個人申請資訊整合」為主題，建立一個主題式網站，  
並提供動態查詢與互動功能，讓使用者能查詢系所與瀏覽大學申請相關資訊。

## 功能說明
- 主題式資訊展示
- 動態資料查詢
- 網頁互動效果
- 後端資料處理（Django）

## 使用技術
- Python 3
- Django
- HTML / CSS / JavaScript

## 環境建置與執行
```bash
pip install -r requirements.txt
python manage.py runserver
```
